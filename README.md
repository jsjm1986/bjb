# 企业AI服务多语言网站

这是一个面向企业AI服务的多语言网站项目，支持中文、英文和日文。

## 项目结构

```
├── index.html          # 主页面
├── css/               # 样式文件目录
│   └── style.css      # 主样式文件
├── js/                # JavaScript文件目录
│   ├── main.js        # 主要逻辑文件
│   └── i18n.js        # 国际化处理文件
└── assets/            # 资源文件目录
    └── images/        # 图片资源目录
```

## 技术栈

- HTML5
- CSS3
- JavaScript (ES6+)
- 响应式设计

## 功能特点

- 多语言支持（中文、英文、日文）
- 现代化UI设计
- 响应式布局
- AI服务展示
- 联系表单
